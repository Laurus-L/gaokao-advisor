✅ 优势
• 就业稳定，国企央企多（电网/中车/中船等）

• 起薪偏低（本科平均6-10k，硕士10-15k）
• 部分方向需下车间/现场，环境一般
• 传统制造业加班也不少
• 晋升周期长，靠资历

📚 学习路线建议

大一·基础

  • 📌 高数、线代、概率论、C语言、Python入门
  • 📌 英语四级600+，开始看英文技术文档

大二·核心

  • 📌 数据结构与算法（重中之重）、数据库、计算机网络
  • 📌 开始刷LeetCode，参加ACM/蓝桥杯
  • 📌 GitHub开源贡献，建立技术博客（英文）

大三·方向

  • 📌 选方向：AI/ML / 后端 / 安全 / 数据科学
  • 📌 学AI工具链：PyTorch/LangChain/AI Agent
  • 📌 大厂暑期实习（字节/腾讯/阿里/微软）
  • 📌 用英语读顶会论文（CVPR/NeurIPS/ACL），你的优势

大四·出路

  • 📌 考研 → 华科/浙大/北航 CS强校
  • 📌 就业 → 互联网大厂/外企/金融IT
  • 📌 出国 → 英语好+CS背景，申美国CS master有优势
